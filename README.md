# Sorting-Script
Скрипт для линукса который сортирует файлы из домашней(пока что) директории в ~/files/ с выбором формата и папки в которую нужно этот формат сортировать

## Принимаемые аргументы:
- `-h` или `--help` - выводит справку
- `-a <category>:<format>` - добавляет формат в список форматов для сортировки по данной категории(если формат уже есть в списке, то старый удаляется)
- `-d <format>` - удаляет формат из списка форматов для сортировки из всех категорий
- `-i <file>` - импортирует настройки из файла
- `--edit-sorting-folder <folder>` - редактирует папку для сортировки файлов
- `--add-origin-folder <path>` - добавляет папку в список папок для сортировки файлов(откуда берутся файлы)
- `--remove-origin-folder <path>` - удаляет папку из списка папок для сортировки файлов
- `--dump-config` - выводит конфигурацию в формате json
- `--dump-config-file <file>` - выводит конфигурацию в файл
- `-r <old-folder>:<new-folder>` - переименовывает папку в списке папок для сортировки файлов и автоматически переносит файлы из старой в новую
- `-R <old-folder>:<new-folder>` - переименовывает папку в списке папок для сортировки файлов
- `-f <folder> <format>` - сортирует файлы с указанным форматом из указанной папки в папку с его категорией
- `-f <folder>` - сортирует все файлы из указанной папки в папку с его категорией


**\\/ -- СТАВИТЬ В КОНЦЕ -- \\/**
- `-n` - отменяет сортировку(если надо) 
- `<формат>` - сортирует файлы с указанным форматом в папку с его категорией
- `<пусто>` - сортирует все файлы в папку `~/files/`

## Примеры использования:
- `sorting.py -a music:m4a` - добавляет формат m4a в список форматов для сортировки по категории music
- `sorting.py -d mp3` - удаляет формат mp3 из списка форматов для сортировки из всех категорий
- `sorting.py -i ~/sorting_formats.json` - копирование файла с форматами сортировки для использования
- `sorting.py -n` - отменяет сортировку
- `sorting.py mp4` - сортирует файлы с форматом mp4 в папку с его категорией
- `sorting.py` - сортирует все файлы в папку `~/files/`
- `sorting.py -a music:m4a -d mp3 -n` - добавляет формат m4a в список форматов для сортировки по категории music, удаляет формат mp3 из списка форматов для сортировки из всех категорий и отменяет сортировку
- `sorting.py -f ~/Downloads` - сортирует все файлы из папки Downloads в папку с его категорией
- `sorting.py -r music:музыка` - изменяет название категории music на музыка и переносит все файлы из папки music в папку музыка
- `sorting.py -R music:музыка` - изменяет название категории music на музыка и больше ничего не делает(все файлы остаются на своих местах, а новые будут добавлены в папку музыка)
